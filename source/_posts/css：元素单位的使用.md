### 1、px

px 像素，绝对长度单位，像素 px 是相对于**显示器屏幕分辨率**而言的

<!--more-->

**特点**

1. IE 无法调整那些使用 px 作为单位的字体大小
2. 国外的大部分网站能够调整的原因在于其使用了 em 或 rem 作为字体单位
3. Firefox 能够调整 px 和 em，rem，但是 96%以上的中国网民使用 IE 浏览器（或内核）

### 2、em

1em = 16px

em 是相对长度单位。相对于当前对象内文本的字体尺寸，如当前对行内文本的字体尺寸未被人为设置，则相对于浏览器的默认字体尺寸

如果某个 p 元素为`font-size:12px` ，在它的内部有个 span 标签，设置`font-size:2em` ，那么这个时候 span 字体的大小为 12\*2 = 24px

**特点**

1. em 的值并不是固定的
2. em 会继承**父级元素**的字体大小

### 3、rem

1rem = html 根元素设定的 font-size 的 px 值（如果没有设置就是 1rem = 16px）

使用 rem 为元素设置字体大小时，仍然是相对大小，但相对的是 HTML 根元素

如果 html 的`font-size:12px` ，在其中的 div 设置为`font-size:2rem` ，则这个 div 的的实际字体大小为 2\*12 = 24px

**特点**

1. rem 是相对单位，总是相对于**根元素< html >**
2. 相对大小和绝对大小的优点于一身
3. 修改根元素就成比例的调整所有字体大小
4. 避免字体大小逐层复合的连锁反应
